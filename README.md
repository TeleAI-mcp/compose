# Docker Compose

[![Build Status](https://img.shields.io/github/actions/workflow/status/docker/compose/ci.yml?branch=main&label=actions&logo=github)](https://github.com/docker/compose/actions?query=workflow%3ACI)

Docker Compose is a tool for running multi-container applications on Docker
defined using the Compose file format.
The full documentation is at [docs.docker.com/compose](https://docs.docker.com/compose).

## Installation and Upgrading

For instruction on installing and upgrading Docker Compose, see [Install Docker Compose](https://docs.docker.com/compose/install/).
